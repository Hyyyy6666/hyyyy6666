### 简介

CSS全名为**cascading style sheets**，中文名**层叠样式表**

用于定义网页样式和布局的样式语言。通过CSS，可以指定页面各个元素的颜色、字体、大小、间距、边框、背景等样式，从而实现更精确的页面设计。

### 语法

CSS通常由选择器、属性和属性值组成，可以多个组合，以便同时应用多个样式：

选择器{

属性1：属性值1;

属性2：属性值2；

}

1. 选择器的声明中可以写无数条属性

2. 声明的每一行属性，都需要以英文分号结尾；

3. 生命中的所有属性值和值都是以键值对这种形式出现：

   //这就是一个p标签的选择器

   p{

   color：blue;

   font-size：16px;

   }



### CSS三种导入方式

1. 内联样式:直接将CSS样式直接放在HTML标签中，在其中直接定义

2. 内部样式表：在HTML文档的head标签定义

3. 外部样式表：单独放在一个文件中，在HTML中通过链接定义进文档使用

   三种导入方式优先级：1>2>3

### 盒子模型

常用于布局的概念

相关属性：

​	内容‘content’：盒子包含的实际内容，比如文本、图片等

​	内边距‘padding’：围绕在内容内部，是内容与边框之间的空间，可使用padding属性来设置

​	边框'border'：围绕在内边距的外部，是盒子的边界。可以使用‘border’属性设置

​	外边距‘margin’：围绕在边框的外部，是盒子与其他元素之间的空间，可使用‘margin’属性设置。

1. 边框样式：实线（solid）虚线（dashed）点线（dotted）双实线（double）
2. border-style、border-width、border-color：仍然为复合属性，通过一个属性，设置四个边框，且遵循上右下左原则（只给一个量四个边框都为同一个值；给两个量上下都是前一个值，左右为另一个值；给出三个值，第一个是上，第二个是左，第三个是下，第四个自己找对应值，即与左的值相同四个量都给时，遵循上右下左）。

3.border-left，针对左边框进行多个量的设置

4.其他属性类比上述2和3

### 浮动

1. 定义：**浮动**属性属于创建浮动边框，将其移动到一边，知道左边缘或右边缘触及包含块或另一个浮动框的边缘，这样即可使得元素进行浮动。

2. 语法：

   选择器{

   float：left/right/none;

   }

   注意：浮动是相对于父元素浮动，只会在父元素的内部移动。

3. 三大特性：

   脱标：脱离标准流，移动到指定位置。

   一行显示，顶部对齐。

   具备行内块元素特性。

   4. 清除浮动常见方法：当父元素坍塌，需要通过清楚浮动来恢复布局

### 定位

1. 与前者浮动相比：定位可以精准布局某元素，但缺乏灵活性；浮动虽然有灵活性，但不容易控制。

2. 方式：

   ​	相对定位：相对于元素在文档流中的正常位置进行定位。

   ​	绝对定位：相对于其最近的已定位祖先元素进行定位，不占据文档流。

   ​	固定定位：相对于浏览器窗口进行定位。不占据文档流，固定在屏幕上的位置，不随滚动而移动。
