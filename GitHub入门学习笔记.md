### Git与GitHub概念学习
1. 学习路径：从【廖雪峰的官网】(https://liaoxuefeng.com/books/git/introduction/index.html)中简单学习了Git和GitHub的概念，
2. 学习内容：
    ①定义：**Git**是一个分布式版本控制系统，**GitHub**则是基于Git的代码托管平台。
    ②作用：**Git**类似一个记录员,可以记录并回溯此前对文件的所有修改，**GitHub**更像是一个共享的云盘，可以上传、收藏且共同修改内容，实现多方合作。

### Git初始化和版本库创建
1. 学习路径：从【廖雪峰的官网】(https://liaoxuefeng.com/books/git/introduction/index.html)

2. 学习内容：
    ①学会简单指令：用**$ git init**建立仓库；用 **$ git add** 和 **$ git commit -m "wrote a readme file"**将文件放进仓库。
    ②值得注意的是：学习指令时要关注空格，例如学习初始化时，指令中未加入空格，导致一直提示找不到指令。
