## 学习内容
### 对git的认识
起初对于git了解不是很多，在认真阅读廖雪峰的官方网站中对于git的介绍后发现其是目前世界上最先进的分布式控制系统，可以多人协同操作，还能查看修改内容。同时对于Linus这个人十分敬佩，实在是太牛了👍。
### git的使用
在最初使用git bash时，看着满屏的命令，当时真的头大，每一个都对我来说十分陌生，只能从头学习。在阅读廖雪峰的git的同时，我发现自己还是不太能够理解≡(▔﹏▔)≡，后面就去询问AI，去小破站找视频，问mentor等。在小破站看了一个视频[给傻子的git教程](https://www.bilibili.com/video/BV1Hkr7YYEh8/?spm_id_from=333.1007.top_right_bar_window_history.content.click&vd_source=a78069782a9aa4e8a2dd04738f67b492)
哇，真的是觉得自己还有很多要学习的地方🥲。
### git学习感受
从下载git，到配置用户信息，再到创建本地仓库，远程推送等，也在一步步学习，从一窍不通，到现在明白git add、git commit等，虽然目前只是浅显的了解，但只要每天进步一点点也很开心😁。
